# Kickstarter Crowdfunding Analysis

## Tools Used
Excel | Power BI | Tableau | MySQL

## Project Overview
This project analyzes Kickstarter crowdfunding data to understand project success rates, category performance, funding trends, and outcomes.

## Key Work
- Converted Epoch Time to Date
- Created Calendar Table
- Built Star Schema Data Model
- Converted Goal Amount to USD
- KPI Analysis by Category, Location, and Time
- Successful Projects Analysis

## Dashboards
- Excel Dashboard
- Power BI Dashboard
- Tableau Dashboard

## SQL Work
- Data Modeling
- KPI Queries
- Trend Analysis
